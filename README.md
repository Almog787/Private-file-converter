# 🧮 Global Calc Hub - Precision Ethereal Design

A high-end, professional web-based suite of financial and mathematical tools. Designed with a focus on speed, precision, and a modern "Ethereal" aesthetic.

**🌐 Live Demo:** [https://almog787.github.io/Private-file-converter/](https://almog787.github.io/Private-file-converter/)

---

## ✨ Features

* **Compound Interest Calculator:** Visualize long-term wealth growth with precision.
* **Percentage Calculator:** Quick tools for discounts, tips, and financial ratios.
* **Unit Converter:** Seamlessly convert between Metric (KG) and Imperial (LBS) units.
* **Privacy First:** All calculations are performed locally in the browser. No data is sent to a server.
* **Responsive Design:** Optimized for Desktop, Tablets, and Mobile devices.
* **Advanced Analytics:** Integrated with GA4 for user engagement tracking.

## 🚀 Technologies Used

* **Tailwind CSS:** For the modern, utility-first UI design.
* **JavaScript (ES6+):** For real-time calculations and event tracking.
* **Google Analytics 4:** To monitor tool popularity and user flow.
* **GitHub Pages:** Fast and reliable hosting.
* **Material Symbols:** For a clean, intuitive iconography.
